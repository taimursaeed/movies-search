### Setup Environment Variables

Setup `.env.local` at root and get API KEY from [https://www.themoviedb.org/](https://www.themoviedb.org/)

`.env.local` example:
```
API_URL=https://api.themoviedb.org/3/search/movie
API_KEY=
```
